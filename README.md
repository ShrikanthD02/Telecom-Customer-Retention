# Telecom Customer Retention Dashboard – 2025
**Author:** Shrikanth Dhangar  
**Contact:** shrikanthdhangar188@gmail.com

## Project Overview
A Power BI dashboard that analyses customer churn and retention for a telecom dataset. It includes KPIs (Total Customers, Churn Rate, Avg CLV), charts (Churn by Contract, Churn Trend by Tenure), and a customer-level table.

## Files
- `data/Cleaned_Telecom_Customer_Retention.csv` — cleaned dataset used in the dashboard.
- `pbix/Telecom_Customer_Retention.pbix` — Power BI project file (if included).
- `images/dashboard_screenshot.png` — dashboard screenshot.
- `theme/TelecomTheme.json` — Power BI theme file (colors/fonts).

## Key Features
- Interactive KPI cards
- Churn Rate by contract, payment method and internet service
- Trend analysis by tenure
- Customer-level table with CLV
- What-if and forecast examples (optional)

## How to run
1. Download the repo.  
2. Open `pbix/Telecom_Customer_Retention.pbix` in Power BI Desktop.  
3. If PBIX not included, import `data/Cleaned_Telecom_Customer_Retention.csv` via **Get Data → Text/CSV**.
